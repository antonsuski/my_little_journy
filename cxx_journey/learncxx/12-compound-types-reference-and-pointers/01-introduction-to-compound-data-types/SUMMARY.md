# Conclusion

There are several compund types in c++:
- Arrays
- Functions
- Pointers
    - Pointers to objects
    - Pointers to functions
- Member pointers
    - Pointers to data member
    - Pointers to function member
- Reference types
    - L-value reference
    - R-value reference
- Enumerated types:
    - Scoped enumeration
    - Unscoped enumeration
- Class types
    - class
    - structure
    - union
